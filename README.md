🍽️ MunchMun

A web platform that helps reduce food waste by allowing users to buy surplus food from restaurants, canteens, and food vendors at discounted prices before it gets thrown away — with a built-in wallet system and secure pickup via code.

📖 Description

FoodSaver connects food providers with users who are looking for affordable meals. Businesses can list leftover food at the end of the day, and users can reserve portions, pay through the platform, and collect their food using a unique pickup code.

This helps:

  🌍 Reduce food waste
  💸 Save money
  🏪 Support local businesses
  ⚡ Enable fast and secure transactions
  

✨ Features:
  
 👤 For Users:

  Sign up and log in
  Browse available food offers
  Reserve food portions
  Pay securely using an in-app wallet
  Receive a unique pickup code after payment
  View pickup time and location
  Track wallet balance and transactions
  
 🏪 For Food Providers:

  Register a business account
  List surplus food items
  Set quantity, price, and pickup window
  Verify orders using pickup codes
  Manage reservations and completed orders
  
 💳 Wallet System:

  Users have a personal wallet inside the platform
  Money can be added to the wallet
  Payments are made instantly when reserving food
  No need for cash at pickup
  
 🔐 Pickup Code System:

  After payment, each order generates a unique code
  Users present the code when picking up their food
  Businesses verify the code before handing over the order
  Prevents fraud and ensures correct order collection
  
 ⚙️ How It Works:

  Food providers upload surplus meals.
  Users browse available offers.
  Users reserve a portion.
  Payment is made from the user’s wallet.
  A unique pickup code is generated.
  Users collect their food and present the code.
