🍽️ MunchMan

A web platform that helps reduce food waste by allowing users to buy surplus food from restaurants, canteens, and food vendors at discounted prices before it gets thrown away — with a built-in wallet system and secure pickup via code.

📖 Description

MunchMan connects food providers with users who are looking for affordable meals. Businesses can list leftover food at the end of the day, and users can reserve portions, pay through the platform, and collect their food using a unique pickup code.

This helps:

  🌍 Reduce food waste
  💸 Save money
  🏪 Support local businesses
  

✨ Features:
  
 👤 For Users:

  Sign up and log in
  Users discover content by presenting suggestions tailored to their preferences
  and food places near them
  Reserve food portions
  Receive a unique code for pick up after making a reservation
  View pickup time and location

  
 🏪 For Food Providers:

  Register a business account
  List surplus food items
  Set quantity, price, and pickup window
  Verify orders using pickup codes
  Manage reservations and completed orders

 🔐 Pickup Code System:

  After making a reservation, each order generates a unique code
  Users present the code when picking up their food
  Businesses verify the code before handing over the order
  Prevents fraud and ensures correct order collection
  
 ⚙️ How It Works:

  Food providers upload surplus meals.
  Users browse available offers.
  Users reserve a portion.
  A unique pickup code is generated.
  Users get their food and present the code.

💚 Save food. Save money. Save the planet.
